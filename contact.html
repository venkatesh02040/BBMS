About This Project

This Blood Bank Management System helps users register with Aadhaar, find nearby donors, and track donation eligibility. It ensures accurate donor records with auto-updated data and 6-month donation cooldown logic. The app is built using pure HTML, CSS, and JavaScript with a beginner-friendly structure and responsive design.
